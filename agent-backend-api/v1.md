# agent-backend /v1 Contract

Base URL: `http(s)://<host>:8080`

### 数据流

#### RAG 数据 Pipeline（统一）

```
所有数据入口（见下方汇总）
    │
    ▼
SHA256 去重检查 (SQLite dedup_records 表)
    │
    ├─→ [已存在] 跳过，报告 existing_cos_key
    │
    ▼ [新内容]
存储 COS (原始文件)
    │
    ▼
记录 SHA256 → dedup_records (cos_key, size)
    │
    ▼
直接 RAG ingest → Qdrant (向量数据库)
    │
    ▼
rag.query / search
```

**所有数据入口（统一走 dedup → COS → Qdrant）：**

| 入口 | 存储格式 | 触发条件 |
|------|---------|---------|
| `data.ingest` | Markdown | 自动 |
| `files.upload` | 用户指定 | `auto_ingest_rag=true` |
| `crawl4ai.page` | Markdown | `auto_ingest_rag=true` |
| Anna's Archive download | 原始二进制 | 自动 |
| `github.batch_download` | 原始二进制 | `store_in_cos=true` |
| `rag.sync_to_repo` | 原始二进制+Markdown | `auto_ingest_rag=true` |

**去重机制：** 基于 SHA256(content) 哈希，同一内容不会重复存储。可用 `overwrite=true`（data.ingest）强制覆盖。

**GitHub 审计跟踪：** `rag.sync_to_repo` 会将 raw 文件和 converted Markdown 写入 `HIT-A/HITA_RagData` 的 `sources/github/<repo>/raw/` 和 `sources/github/<repo>/converted/` 目录，作为本地审计记录，不影响线上 Qdrant。

**注意：** `rag.intake_manual_folder` 已废弃，不再需要。

### Data Skills

#### `data.ingest` - 统一数据接入（异步）

**输入**
```json
{
  "source_type": "manual",
  "source_name": "课程笔记",
  "content": "# 笔记内容...",
  "store_in_cos": true,
  "cos_prefix": "rag-content",
  "auto_ingest_rag": true,
  "overwrite": false
}
```

**参数说明：**

| 参数 | 类型 | 说明 |
|------|------|------|
| `source_type` | string | `manual` / `github` / `annas` / `arxiv` / `crawl` |
| `source_name` | string | 来源名称（论文ID、文件名等） |
| `content` | string | 文本内容（manual/github/crawl） |
| `content_base64` | string | base64 编码内容 |
| `file_path` | string | 本地文件路径 |
| `source_url` | string | 下载来源 URL（annas/arxiv） |
| `store_in_cos` | bool | 是否存储到 COS（默认 true） |
| `cos_prefix` | string | COS 路径前缀（默认 `rag-content`） |
| `auto_ingest_rag` | bool | 是否直接 ingest 到 Qdrant（默认 true） |
| `overwrite` | bool | 是否覆盖已存在的相同 SHA256 内容 |

**输出（同步返回）**
```json
{
  "ok": true,
  "output": {
    "status": "completed",
    "file_count": 1,
    "chunks_count": 5,
    "cos_keys": ["rag-content/manual/a1b2c3_课程笔记.md"],
    "vectorized": true,
    "warnings": []
  }
}
```

**重复内容时（SHA256 已存在）：**
```json
{
  "status": "skipped",
  "cos_keys": ["rag-content/manual/a1b2c3_课程笔记.md"],
  "warnings": ["duplicate SHA256: a1b2c3d4... (existing COS: rag-content/manual/a1b2c3_课程笔记.md)"]
}
```

**COS 路径格式：** `{cos_prefix}/{source_type}/{sha256[:12]}_{source_name}.md`

#### `crawl4ai.page` - 抓取单页

**输入**
```json
{
  "url": "https://example.com/article",
  "wait_for": "",
  "exclude_paths": ["/api/", "/admin/"],
  "content_filter": true,
  "output_format": "markdown",
  "queue_to_intake": true,
  "intake_repo": "HIT-A/HITA_RagData",
  "intake_branch": "main",
  "intake_skill_dir": "incoming/crawl_raw"
}
```

**输出**
```json
{
  "ok": true,
  "output": {
    "markdown": "# Article Title\n\n文章内容...",
    "html": "<html>...</html>",
    "intake": {
      "enabled": true,
      "queued": true,
      "path": "incoming/crawl_raw/..."
    }
  }
}
```

### 数据流

#### RAG 数据 Pipeline

```
用户上传文档
    │
    ▼
data.ingest / files.upload
    │
    ├─→ COS Storage (永久存储)
    │
    └─→ GitHub Raw Repo (入队)
            │
            ▼
    rag.intake_manual_folder (定时/手动)
            │
            ├─→ Parse Document (markdown)
            │
            ├─→ Embedding (BigModel API)
            │
            └─→ Qdrant (向量数据库)
                    │
                    ▼
            rag.query / search
```

#### Search 聚合 Pipeline

```
用户查询
    │
    ▼
search (聚合入口)
    │
    ├─→ rag (Qdrant 向量搜索)
    │
    ├─→ brave (Brave Search API)
    │
    ├─→ github (GitHub Code Search API)
    │
    ├─→ arxiv (arXiv MCP)
    │
    ├─→ annas (Anna's Archive MCP)
    │
    ├─→ cos (COS 文件列表)
    │
    ├─→ course (pr-server 课程搜索)
    │
    ├─→ course_read (pr-server README 读取)
    │
    └─→ hit_teacher (crawl4ai 爬取教师主页)
            │
            ▼
    aggregator.summarize (LLM 总结)
            │
            ▼
        最终结果
```

### 外部依赖

| 服务 | 用途 | 环境变量 |
|------|------|---------|
| 腾讯云 COS | 文件存储 | `COS_SECRET_ID`, `COS_SECRET_KEY`, `COS_BUCKET`, `COS_REGION` |
| Qdrant | 向量数据库 | `QDRANT_URL`, `QDRANT_API_KEY` |
| BigModel API | 文本嵌入（优先）/ 总结回退 | `BIGMODEL_API_KEY` |
| MiniMax API | 总结（优先） | `MINIMAX_API_KEY`, `MINIMAX_MODEL` |
| pr-server | 课程/PR 服务 | `PR_SERVER_URL`, `PR_SERVER_TOKEN` |
| Brave Search | 网页搜索 | `BRAVE_API_KEY` |
| GitHub | 代码仓库 / 审计跟踪 | `GITHUB_TOKEN` |
| crawl4ai MCP | 网页爬取 | Python 环境 |
| unstructured MCP | 文档转换（PDF/PPTX/DOCX→MD） | Python 环境 |
| Anna's Archive MCP | 文献下载 | `ANNAS_SECRET_KEY` |
| arXiv MCP | 论文搜索/下载 | Python 环境 |
